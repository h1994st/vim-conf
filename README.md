Vim Configuration
===

## Getting Started

Only for linux

#### via curl

```bash
sh -c "$(curl -fsSL https://raw.githubusercontent.com/h1994st/interesting-scripts/master/sync-vim-conf.sh)"
```

#### via wget

```bash
sh -c "$(wget https://raw.githubusercontent.com/h1994st/interesting-scripts/master/sync-vim-conf.sh -O -)"
```
